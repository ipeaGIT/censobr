# Download file from url

Download file from url

## Usage

``` r
download_file(
  file_url = parent.frame()$file_url,
  showProgress = parent.frame()$showProgress,
  cache = parent.frame()$cache,
  verbose = parent.frame()$verbose
)
```

## Arguments

- file_url:

  String. A url.

- showProgress:

  Logical.

- cache:

  Logical.

- verbose:

  Logical.

## Value

A string to the address of the file
