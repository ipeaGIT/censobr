# Get path to cache directory for censobr files

Get the path to the cache directory currently being used for for the
censobr files

## Usage

``` r
get_censobr_cache_dir()
```

## Value

Path to cache dir

## See also

Other Cache data:
[`censobr_cache()`](https://ipeagit.github.io/censobr/dev/reference/censobr_cache.md),
[`set_censobr_cache_dir()`](https://ipeagit.github.io/censobr/dev/reference/set_censobr_cache_dir.md)

## Examples

``` r
# get path to cache directory
get_censobr_cache_dir()
#> [1] "/home/runner/.cache/R/censobr"
```
